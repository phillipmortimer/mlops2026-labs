# PDPSL Labs

This repository contains the practical lab sessions for the revised PDPSL course, covering MLOps, LLMOps, and AgentOps.

## Labs

- `lab-01-mlops`: MLOps foundations with MLflow
- `lab-02-llmops`: LLMOps foundations with Langfuse tracing and evals
- `lab-03-agentops`: AgentOps foundations with reusable skills and plugins

## Working With A Lab

Each lab is intended to be self-contained. Open the lab directory and follow its `README.md`.
